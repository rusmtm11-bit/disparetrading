# Dispare Trading — сайт

Статический многостраничный сайт (HTML/CSS/JS, без сборки и зависимостей).

## Структура
- `index.html` — главная
- `catalog.html` — каталог категорий
- `delivery.html` — оплата и доставка
- `warranty.html` — гарантии и возврат
- `about.html` — о компании
- `contacts.html` — контакты и форма заявки
- `privacy.html` — политика конфиденциальности (152-ФЗ, требует юридической проверки и заполнения реквизитов)
- `assets/` — стили, скрипты, логотип

## Форма заявки
Пока не подключена намеренно — сбор персональных данных (имя/телефон/email) через форму на сайте требует соблюдения 152-ФЗ, включая размещение обработчика на сервере в РФ. Сейчас все кнопки «Связаться с нами» ведут на блок с прямыми контактами (телефон/почта) на странице `contacts.html`. Страница `privacy.html` (политика конфиденциальности) уже подготовлена как шаблон — когда форма понадобится, её не придётся переделывать с нуля, но перед подключением формы текст и техническую реализацию стоит ещё раз сверить с юристом.

## Перед публикацией обязательно
1. Заполнить реквизиты компании в `contacts.html` и `privacy.html` (сейчас там плейсхолдеры вида `[уточняется]`).
2. Юридически проверить текст `privacy.html`.
3. Если меняете хостинг (например, на Cloudflare Pages/Netlify), сначала настроить деплой оттуда, а уже потом делать репозиторий на GitHub приватным — см. пояснение в чате.

## Локальный просмотр
Просто откройте `index.html` в браузере, либо запустите локальный сервер:
```
python3 -m http.server 8000
```
